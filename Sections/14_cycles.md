# Розділ 14. Цикли в системі

Інформаційна система інтернет-магазину містить кілька типів циклів, що впливають на її роботу та динаміку бізнес-процесів.

## 14.1 Підсилюючі цикли (Reinforcing loops)

- **Цикл продажів:** збільшення продажів -> більше відгуків і рейтингів -> підвищення довіри -> ще більше продажів.
- **Цикл користувачів:** більше зареєстрованих користувачів -> активність на платформі -> привернення нових користувачів.

## 14.2 Балансуючі цикли (Balancing loops)

- **Цикл запасів:** зменшення товарів на складі -> поповнення запасів -> стабілізація наявності.
- **Цикл обробки замовлень:** надходження замовлень -> обробка та виконання -> контроль завантаженості системи.

## 14.3 Інші цикли

- **Фінансовий цикл:** надходження коштів -> облік у системі -> формування звітності -> контроль витрат.
- **Цикл обслуговування клієнтів:** звернення користувача -> підтримка -> вирішення проблем -> підвищення лояльності.
