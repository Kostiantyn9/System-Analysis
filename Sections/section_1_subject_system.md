# Розділ 1. Вибір предметної системи для проєктування

## 1.1 Опис предметної системи

Для проєктування обрано **інформаційну систему інтернет-магазину**.  
Система призначена для автоматизації процесів онлайн-продажу товарів, управління замовленнями та взаємодії з клієнтами через веб-інтерфейс.

Метою створення системи є забезпечення повного циклу електронної комерції:

- представлення товарів у каталозі
- оформлення та обробка замовлень
- управління користувачами
- контроль продажів

Предметна область охоплює електронну торгівлю, облік товарів, обробку замовлень та управління клієнтами.

Очікувані користувачі системи:

- покупці інтернет-магазину
- адміністратори системи
- неавторизовані відвідувачі

---

## 1.2 Обґрунтування вибору системи

Вибір інформаційної системи інтернет-магазину обґрунтовано наступними факторами:

### Актуальність

Інтернет-магазини є одним із найбільш поширених типів інформаційних систем у сучасному цифровому середовищі. Онлайн-торгівля активно розвивається та потребує ефективних програмних рішень для автоматизації бізнес-процесів.

### Складність

Система включає різні функціональні компоненти:

- управління товарами
- обробку замовлень
- роботу з користувачами
- контроль доступу
- збереження та обробку даних

Це дозволяє розглянути повний цикл проєктування інформаційної системи.

### Доцільність автоматизації

Автоматизація процесів інтернет-магазину дозволяє:

- зменшити кількість ручних операцій
- підвищити швидкість обробки замовлень
- знизити ризик помилок
- покращити якість обслуговування клієнтів

Таким чином, обрана предметна система є реалістичною, актуальною та придатною для моделювання.
