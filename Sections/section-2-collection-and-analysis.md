 Розділ 2. Збір і аналіз вимог

## 2.1 Обстеження предметної області

Інформаційна система інтернет-магазину призначена для автоматизації процесів онлайн-продажу товарів. Основні бізнес-процеси:

1. Управління каталогом товарів
   - додавання, редагування, видалення товарів
   - управління категоріями
   - відображення товарів користувачам

2. Оформлення замовлення
   - додавання товарів у кошик
   - введення даних доставки
   - підтвердження замовлення

3. Оплата замовлення
   - вибір способу оплати
   - підтвердження платежу
   - зміна статусу замовлення

4. Управління користувачами
   - реєстрація
   - авторизація
   - редагування профілю

5. Обробка замовлень адміністрацією
   - перегляд замовлень
   - зміна статусів
   - облік продажів

---

## 2.2 Функціональні вимоги

Система повинна забезпечувати:

- реєстрацію та авторизацію користувачів
- перегляд каталогу товарів
- пошук та фільтрацію товарів
- управління кошиком
- оформлення замовлення
- вибір способу оплати
- перегляд історії замовлень
- адміністративне управління товарами
- управління замовленнями
- управління користувачами

---

## 2.3 Нефункціональні вимоги

1. Продуктивність
   - система повинна обробляти запити користувачів без значних затримок

2. Безпека
   - автентифікація користувачів
   - захист персональних даних
   - розмежування прав доступу

3. Надійність
   - збереження даних при збоях
   - резервне копіювання

4. Масштабованість
   - можливість роботи при збільшенні кількості користувачів

5. Зручність використання
   - інтуїтивний інтерфейс
   - доступність через браузер

---

## 2.4 Обмеження та зовнішні умови

- система працює через веб-браузер
- потрібне підключення до Інтернету
- база даних використовується для збереження інформації
- доступ до системи здійснюється через облікові записи

---

## 2.5 Специфікація вимог (SRS)

Система повинна забезпечити повний цикл онлайн-продажу товарів: від перегляду каталогу до обробки замовлення.  
Доступ до функцій визначається роллю користувача.  
Система повинна відповідати вимогам безпеки, продуктивності та надійності.
